# PJSK 歌词猜曲插件

## 功能简介

本插件是一款专注于《初音未来 缤纷舞台》的歌词猜曲游戏插件。用户根据歌词片段猜出歌曲名称

歌曲资源截至日服5.5周年前

## 指令列表

### 游戏指令

| 指令               | 别名                             | 说明               |
| ---------------- | ------------------------------ | ---------------- |
| `歌词猜曲`           | `pjsk歌词猜曲`、`猜歌词`、`歌词识曲`、`歌词猜歌` | 开始一轮歌词猜曲游戏       |
| `歌词猜曲分数`         | `pjsk歌词猜曲分数`、`猜歌词分数`           | 查看自己的游戏数据统计      |
| `歌词猜曲排行榜`        | `歌词猜曲排行`、`pjsk歌词猜曲排行榜`         | 查看总分排行榜          |
| `歌词猜曲自定义名称 [名称]` | `歌词猜曲昵称`、`歌词猜曲名称`              | 设置自定义ID（不带参数可清除） |

### 帮助与管理

| 指令       | 说明                 |
| -------- | ------------------ |
| `歌词猜曲帮助` | 显示帮助信息             |
| `刷新本地数据` | 重新加载本地翻译和别名数据（管理员） |

## 配置说明

| 配置项                     | 类型   | 默认值 | 说明               |
| ----------------------- | ---- | --- | ---------------- |
| `answer_timeout`        | int  | 30  | 答题超时时间（秒）        |
| `daily_play_limit`      | int  | 10  | 每日游戏次数上限（-1为无限制） |
| `game_cooldown_seconds` | int  | 30  | 游戏冷却时间（秒）        |
| `ranking_display_count` | int  | 10  | 排行榜显示的最大人数       |
| `super_users`           | list | \[] | 管理员用户ID列表        |
| `blacklist`             | list | \[] | 黑名单用户ID列表        |
| `group_whitelist`       | list | \[] | 群聊白名单（为空则所有群可用）  |


## 游戏玩法

1. 发送 `歌词猜曲` 开始游戏
2. 系统会随机选择一首歌曲，展示20行连续的歌词片段
3. 同时显示10个选项（包含正确答案）
4. 在限时内输入数字(1-10)选择你认为正确的答案
5. 答对获得1分，答错或超时不计分
